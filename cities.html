In bootstrap 5 minimum length for the col classes as follows:
col: min length is 0px (xs)
col-sm: min length is 576px
col-md: min length is - 768px
col-lg: min length is - 992px
col-xl: min length is - 1200px
col-xxl: min length is - 1400px

when the screen width is less than the minimum length of the respective class then the columns will stack vertically. In our code we used "col-sm" whose min length is 576px. When screen width is reduced than 576px, columns will stack vertically. Similarly for "col-md" whose min length is 768px. When screen width is reduced than 768px, columns will stack vertically.
